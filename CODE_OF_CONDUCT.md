# Code of Conduct

## Our Standard

This project expects respectful, professional collaboration from everyone participating in its community.

Examples of expected behavior:

- Use welcoming and inclusive language.
- Give and receive technical feedback in good faith.
- Assume mistakes can be corrected without hostility.
- Focus discussion on the code, the behavior, and the evidence.
- Respect different experience levels, backgrounds, and perspectives.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Discriminatory language or conduct.
- Trolling, deliberate intimidation, or sustained disruption.
- Publishing someone else's private information without consent.
- Sexualized language, imagery, or unwanted advances.

## Scope

This Code of Conduct applies in all project spaces, including:

- GitHub issues, pull requests, and discussions
- documentation and code review threads
- any public or private communication conducted in relation to this project

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards. They may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

Maintainers may take any action they consider appropriate in response to unacceptable behavior, including warnings, temporary bans, or permanent bans from project spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers through the repository owner or maintainers listed on GitHub. Include links, screenshots, or any other context that helps explain what happened.

All reports will be reviewed promptly and handled as confidentially as reasonably possible.
